# Volume Blog Editor - Setup Guide

## Prerequisite: Install Node.js
To run the offline editor that auto-saves your files, you need a small program called **Node.js**.

1. Go to [https://nodejs.org/](https://nodejs.org/)
2. Download the **LTS Version** (Recommended for Most Users).
3. Install it (just click Next > Next > Finish).

## How to Run the Editor
1. Inside the `volume-blog` folder, double-click **`run_editor.bat`**.
2. A black window will appear (this is the local server).
3. Your browser will automatically open to the Editor.
4. **Keep the black window open** while you are editing.

## How to Publish
1. After you save your posts in the editor, the files in `volume-blog` are updated.
2. Upload the contents of the `volume-blog/public` folder to your website (`volume-app.net`).
